# 프로젝트 관리 트래커 스킬

## 1. WBS(Work Breakdown Structure) 작성

### 1.1 WBS 표준 구조

```
프로젝트 (Level 0)
├── 1. 주요 단계 (Level 1)
│   ├── 1.1 산출물/작업 그룹 (Level 2)
│   │   ├── 1.1.1 세부 작업 (Level 3 = Work Package)
│   │   ├── 1.1.2 세부 작업
│   │   └── 1.1.3 세부 작업
│   └── 1.2 산출물/작업 그룹
│       ├── 1.2.1 세부 작업
│       └── 1.2.2 세부 작업
├── 2. 주요 단계
│   └── ...
└── 3. 주요 단계
    └── ...
```

## 3. 진행률 대시보드

### 3.1 전체 현황

```markdown
## 프로젝트 대시보드: [프로젝트명]
최종 업데이트: ____년 __월 __일

### 전체 진행률
████████████████░░░░ 80% (16/20 작업 완료)

### 단계별 현황
| 단계 | 진행률 | 상태 |
|------|--------|------|
| 기획 | ████████████████████ 100% | ✅ 완료 |
| 설계 | ████████████████████ 100% | ✅ 완료 |
| 개발 | ████████████░░░░░░░░ 60% | 🔄 진행중 |
| 테스트 | ░░░░░░░░░░░░░░░░░░░░ 0% | ⏳ 대기 |
| 오픈 | ░░░░░░░░░░░░░░░░░░░░ 0% | ⏳ 대기 |

### 핵심 지표
- 예정 완료일: ____.__.__
- 예상 완료일: ____.__.__  (지연: __일)
- 잔여 작업: __건
- 이슈: __건 (미해결: __건)
```

### 3.2 주간 진행 보고

## 5. RACI 매트릭스 (역할 책임 배분)

```
R=Responsible(실행), A=Accountable(최종책임), C=Consulted(자문), I=Informed(통보)

| 작업 \ 담당 | PM | 기획팀 | 개발팀 | 디자인팀 | QA | 경영진 |
|-------------|-----|--------|--------|----------|-----|--------|
| 요구사항 정의 | A | R | C | C | I | I |
| 화면 설계 | A | C | C | R | I | I |
| 개발 | A | I | R | C | I | I |
| 테스트 | A | I | C | I | R | I |
| 최종 승인 | C | I | I | I | I | A |
```

## 6. 출력 형식

기본 출력은 `.xlsx` (Excel) 파일이며, 시트 구성은 아래와 같다.

### 6.1 Excel 파일 시트 구성

| 시트명 | 내용 | 주요 열 |
|--------|------|---------|
| **WBS** | 업무분류체계 + 일정 | WBS ID, 작업명, 담당자, 시작일, 종료일, 기간, 선행작업, 산출물, 상태 |
| **간트차트** | 조건부 서식 기반 일정 시각화 | 작업명, 담당자, 날짜 열(주 단위) → 셀 색상으로 바 표현 |
| **대시보드** | 진행률 요약 | 단계, 전체 작업 수, 완료 수, 진행률(%), 상태 |
| **이슈 트래커** | 이슈 목록 + 상태 관리 | 이슈 ID, 제목, 우선순위, 카테고리, 담당자, 기한, 상태, 조치 내용 |
| **RACI** | 역할 책임 매트릭스 | 작업 \ 담당자별 R/A/C/I |

### 6.2 Excel 서식 규칙

```
상태 조건부 서식:
  ✅완료   → 셀 배경 #D4EDDA (연한 초록)
  🔄진행중 → 셀 배경 #FFF3CD (연한 노랑)
  ⚠️지연   → 셀 배경 #F8D7DA (연한 빨강)
  ⏳대기   → 셀 배경 #E2E3E5 (연한 회색)

우선순위 조건부 서식:
  🔴긴급 → 글자색 빨강 + Bold
  🟠높음 → 글자색 주황
  🟡보통 → 글자색 기본
  🟢낮음 → 글자색 회색

헤더행: 배경 #2D3748, 글자색 #FFFFFF, Bold, 가운데 정렬
데이터행: Noto Sans KR Regular 10pt, 세로 가운데 정렬
열 너비: 내용에 맞게 자동 조절 (autoWidth)
```

### 6.3 간트차트 시트 상세

```
- A열: 작업명, B열: 담당자
- C열~: 주 단위 날짜 헤더 (3/1주, 3/2주, ...)
- 작업 기간에 해당하는 셀을 조건부 서식으로 색상 채움
  - 정상 진행: #4A7BF7 (파란색)
  - 지연:      #FF6B6B (빨간색)
  - 완료:      #48BB78 (초록색)
- 마일스톤: ◆ 기호 + Bold
```

### 6.4 기타 출력 형식

사용자가 별도 요청 시 아래 형식으로도 출력 가능:
- `.md`: Markdown (텍스트 기반 간이 공유용)
- 텍스트: 대화 내에서 바로 확인
